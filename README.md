# 🖼️ Image Processing Filters Toolkit

## 📌 Overview

This project is a practical implementation of the most important **Image Processing filters and techniques** commonly used in computer vision.

It serves as a hands-on toolkit to understand how images are transformed, enhanced, and analyzed using different filtering methods.

---

## 🚀 Key Features

### 🎨 1. Basic Image Transformations

* Grayscale conversion
* Image resizing
* Image rotation

---

### 🧼 2. Smoothing Filters (Noise Reduction)

* Gaussian Blur
* Median Filter
* Average Filter

📌 Used to reduce noise and smooth images

---

### 🔍 3. Edge Detection

* Sobel Filter
* Canny Edge Detection
* Laplacian Filter

📌 Used to detect boundaries and object edges

---

### 🌗 4. Thresholding Techniques

* Binary Threshold
* Adaptive Threshold
* Otsu Threshold

📌 Used for segmentation and object separation

---

### 🎯 5. Advanced Processing

* Morphological Operations:

  * Erosion
  * Dilation
* Image sharpening
* Contrast enhancement

---

## 🛠️ Tech Stack

* Python
* OpenCV
* NumPy
* Matplotlib

---

## 🧠 What I Learned

* How different filters affect image quality
* Trade-off between noise reduction and detail preservation
* Edge detection techniques and their real-world applications
* Fundamentals of image segmentation
